# ChannelSetting

Use the ChannelSetting component to show a list of channels or create a channel

Similar to ChannelList and channel, the ChannelSetting component must also be wrapped inside SendbirdProvider for it to work well. Specify the height of your wrapper component and ChannelList will assume this height. If you want to adjust width or other visual properties, you might need to do it using CSS

## Usage

```js
import { ChannelSetting, SendbirdProvider, ModalRoot, MenuRoot } from "sendbird-uikit-react";

const App = () => {
  <SendbirdProvider>
    <div style={{ height: '400px' }}>
      <ChannelSetting />
    </div>
    <ModalRoot />
    <MenuRoot />
  </SendbirdProvider>
}
```

## Available Props

```js
{
  onCloseClick: Function, <callback for clicking close icon>
  channelUrl: String, <Url of current channel>
}
```
