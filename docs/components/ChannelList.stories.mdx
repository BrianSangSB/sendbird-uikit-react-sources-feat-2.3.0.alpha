# ChannelList

The ChannelList component must be wrapped inside SendbirdProvider for it to work well. If you place SendbirdProvider on the top level, you can use  ChannelList anywhere in your application without worrying about hierarchies. Please note that  ChannelList has no minimum height and takes the height of the container, so it is recommended to specify the height of the wrapper element to fit your needs

## Usage

```js
import { ChannelList, SendbirdProvider, ModalRoot, MenuRoot } from "sendbird-uikit-react";

const App = () => {
  <SendbirdProvider>
    <div style={{ height: '400px' }}>
      <ChannelList />
    </div>
    <ModalRoot />
    <MenuRoot />
  </SendbirdProvider>
}
```
