Sendbird UI Kit React is an opinionated dropin solution to bring the already potent Javascript SDK to your react webapp

## Benefits

  * Easy to use drop in solution
  * Composable
  * No need to worry about SDK state management, we will take care of it for you
  * Internal state is extendable using HOC pattern(https://reactjs.org/docs/higher-order-components.html)
  * Dark theme support
  * Styles are customizable using CSS on top of theme support
