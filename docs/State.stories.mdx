# Internal Application State

Our internal application state will look like this:

```js
{
  stores: {
    // Keeps the SDK and its life cycle
    sdkStore: {
      initialized: PropTypes.bool,
      loading: PropTypes.bool,
      // We will keep an intialized version of sendbird SDK https://sendbird.github.io/core-sdk-javascript/module-sendbird.html
      sb: PropTypes.shape(SDK),
    },
    // Saves the information related to the user
    userStore: {
      initialized: PropTypes.bool,
      loading: PropTypes.bool,
      // https://sendbird.github.io/core-sdk-javascript/module-model_user-User.html
      user: PropTypes.shape(CurrentUser),
    },
  },
  // these are for internal use for now, should be used to update values inassociated stores
  dispatchers: {
    sdkDispatcher,
    userDispatcher,
  },
  config: { // these are the information that you passed down to the SendbirdProvider
    userId: PropTypes.String,
    appId: PropTypes.String,
    accessToken: PropTypes.String,
    theme: PropTypes.String<light|dark>,
    userListQuery: PropTypes.func,
  },
}
```
